# Video to MP3 Converter 🎥 ➡️ 🎵

โปรแกรมสำหรับแปลงไฟล์วิดีโอเป็นไฟล์เสียงนามสกุล `.mp3` ใช้งานง่ายผ่านหน้าต่าง GUI รองรับไฟล์วิดีโอหลายประเภท

## ✨ คุณสมบัติ (Features)
- **ใช้งานง่าย (User-Friendly)**: มีหน้าต่างโปรแกรม (GUI) ไม่ต้องพิมพ์คำสั่งใน Terminal
- **รองรับหลายนามสกุล**: แปลงได้ทั้ง `.mp4`, `.avi`, `.mkv`, `.mov`, `.flv`, `.wmv`, `.webm`, `.mpg`, `.mpeg`
- **เลือกได้หลายไฟล์**: สามารถเลือกไฟล์วิดีโอหลายๆ ไฟล์เพื่อแปลงพร้อมกันในครั้งเดียว (Batch Conversion)
- **ระบบ Auto-Default**: เมื่อเลือกไฟล์วิดีโอ โปรแกรมจะตั้งค่าโฟลเดอร์ปลายทางให้อัตโนมัติ (โฟลเดอร์เดียวกับวิดีโอต้นฉบับ)
- **แถบสถานะ (Progress Bar)**: แสดงความคืบหน้าการทำงานแบบเรียลไทม์

## 🚀 วิธีใช้งาน (How to use)
1. รันไฟล์ `Video_to_MP3_Converter.bat` (หรือรัน `python convert_all.py`)
2. กดปุ่ม **"Browse Files"** เพื่อเลือกไฟล์วิดีโอที่ต้องการแปลง
3. (ไม่บังคับ) กดปุ่ม **"Browse Folder"** หากต้องการเปลี่ยนที่เก็บไฟล์ผลลัพธ์
4. กดปุ่ม **"เริ่มแปลงไฟล์ (Start Conversion)"**
5. รอจนกระทั่งระบบแจ้งเตือนว่า "งานเสร็จสิ้น"

## 🛠️ สิ่งที่ต้องมี (Prerequisites)
- [Python 3.x](https://www.python.org/)
- [FFmpeg](https://ffmpeg.org/) (จำเป็นสำหรับ `moviepy`)
- ไลบรารี `moviepy`
  ```bash
  pip install moviepy
  ```

## 📂 โครงสร้างโปรเจกต์
- `convert_all.py`: ไฟล์โปรแกรมหลัก
- `Video_to_MP3_Converter.bat`: ไฟล์สำหรับเปิดโปรแกรมได้รวดเร็ว
- `.gitignore`: ไฟล์สำหรับขจัดไฟล์ที่ไม่จำเป็นออกจากระบบ Git

## 💡 หมายเหตุ (Notes)
- โปรเจกต์นี้ได้รับการคลีนเพื่อลบไฟล์ขนาดใหญ่และโฟลเดอร์ที่ไม่ได้ใช้งานออกแล้ว
- ไฟล์ผลลัพธ์ `.mp3` จะถูกสร้างขึ้นในโฟลเดอร์ที่คุณเลือก หรือโฟลเดอร์เดียวกับวิดีโอต้นฉบับโดยค่าเริ่มต้น
